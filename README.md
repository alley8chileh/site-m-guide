# site-m-guide

## 项目简介

`site-m-guide` 是一个用于归档与发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态页面的集合与索引，方便维护与分发。

## 目录说明

```
site-m-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 页面索引（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，页面间互不依赖。
- `assets/`：存放多个页面共用的样式或脚本，按需引用。
- `index.html`：可选的首页，用于列出所有归档页面链接。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，可直接通过浏览器打开使用。  
每个页面均包含完整的结构与样式，不依赖外部服务或第三方资源（除非在页面内显式引用）。

归档目的包括但不限于：
- 保存历史版本的页面快照
- 提供离线可用的页面副本
- 作为静态展示或参考素材

## 维护说明

- 新增页面时，请在 `pages/` 下创建对应文件或目录，并保持命名清晰。
- 若页面依赖公共资源，请将资源放入 `assets/` 并注明版本。
- 不建议在页面中插入跟踪代码、外部脚本或动态内容，以保持归档的纯净性。
- 如需更新已有页面，请保留旧版本或通过 Git 历史追溯。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可将本仓库部署到任意静态托管服务（如 GitHub Pages）以在线访问。
